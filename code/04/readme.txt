Api Rest de Espías y Perritos en el parque

Parte A:
- La API debe recibir los datos de una persona.
- Si al persona es espía, el perrito debe morderla
- Si la persona no es espía, el perrito debe lamerlo

Parte B:
- Una vez logrado el punto anterior, guardar el listado de espías y crear un endpoint para obtener este

Parte C:
Dejar un registro de los eventos generados entre perritos y personas. Ejemplo:
 - "El espía X fue mordido por el perrito Y"
